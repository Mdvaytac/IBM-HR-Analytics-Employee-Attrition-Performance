# HR Attrition SQL Analysis

## Overview

This project analyzes employee attrition data using SQL. The objective is to explore workforce trends, salary distribution, employee satisfaction, overtime impact, and factors related to employee turnover.

### Dataset

**hr_attrition** – Contains employee demographic, job, salary, performance, satisfaction, and attrition information.

## Analyses Performed

* Employee count and average experience by job role
* Salary category distribution (Low, Medium, High)
* Work-life balance analysis
* Employees earning above average salary
* Role stability vs attrition analysis
* Highest-paid employee in each department
* Overtime and attrition relationship
* Salary comparison against department averages
* Departments with the highest attrition rates
* High-risk employee identification
* Salary quartile and job satisfaction analysis
* Salary gap from department maximum
* Top employees by salary hike percentage
* Salary comparison using LAG window function
* Experience level (Junior, Mid, Senior) salary analysis
* Department attrition vs company average
* Employee risk segmentation
* Longest manager-employee relationships by role
* High earners above department average
* Multi-company experience and attrition analysis
* Commute distance and attrition relationship
* Stock option participation analysis
* Department payroll contribution analysis
* New vs experienced employee attrition comparison
* Department and gender workforce analysis
* Promotion delay and attrition analysis
* Attrition trends by years at company
* High-potential employee analysis

## SQL Skills Demonstrated

* Data Aggregation (`COUNT`, `AVG`, `SUM`, `MAX`)
* Conditional Logic (`CASE WHEN`)
* Subqueries
* Common Table Expressions (CTEs)
* Window Functions (`RANK`, `LAG`, `LEAD`, `NTILE`)
* Department-Level Analysis
* Attrition Metrics
* Workforce Segmentation
* Trend Analysis


## Conclusion

This project demonstrates how SQL can be used to analyze employee behavior, identify attrition risks, evaluate compensation patterns, and generate workforce insights that support human resource decision-making.
